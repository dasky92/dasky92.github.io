# Emacs Tips


## Dired

| key     | description                                   |
|---------|-----------------------------------------------|
| h,j,k,l | move with left, up, down, right               |
| enter   | goto directory                                |
| -       | goto back last level directory                |
| +       | create directory                              |
| d       | mark deletion, x to delete                    |
| SPC .   | create file or find file in current directory |
| U       | unselect all                                  |
| m       | mark a specific file or directory             |
| C       | copy to another window                        |
| R       | move to another window                        |
| M       | chmod file                                    |
| O       | chown file                                    |
| i       | edit file or dir                              |
